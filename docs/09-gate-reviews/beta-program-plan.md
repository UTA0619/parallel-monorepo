# Phase 3 — Closed Beta Program Plan

> Owner: Product Agent + Growth Agent  
> Duration: 6 months (Phase 3 timeline)  
> Target: 2,500 users across 5 cohorts

## Cohort Definitions

| Cohort | Persona | Target N | Key Hypothesis |
|--------|---------|----------|----------------|
| 1 — Optimizers | High-achievers seeking peak performance | 500 | D30 retention driven by Affection Score gamification |
| 2 — Seekers | Self-discovery focused, 25–35 | 500 | Retention driven by pattern recognition feature |
| 3 — Parents | Overwhelmed parents needing clarity | 500 | Morning Report open rate ≥70% if timed at 6am |
| 4 — Executives | Decision-heavy roles, 35–50 | 500 | Premium conversion highest, D30 ≥75% |
| 5 — JP Market | Japanese professionals | 500 | Localization quality gate; retention ≥65% |

## Sprint Structure (Per Cohort)

```
Week 1–2: Onboard 500 users, baseline metrics
Week 3–4: Sprint 1 — Onboarding friction reduction
Week 5–6: Sprint 2 — Morning Report personalization improvements  
Week 7–8: Sprint 3 — Retention hooks (streak, progress visualization)
Week 9–10: D30 analysis + NPS interviews
Week 11–12: Iteration or cohort wrap-up + Gate 3 package prep
```

## Recruitment Channels
- Waitlist (landing page): target 10,000 sign-ups before beta opens
- LinkedIn outreach: Cohort 4 (Executives)
- Twitter/X + Reddit (r/selfimprovement, r/productivity): Cohorts 1, 2
- JP channels: Voicy, note.com, LinkedIn JP: Cohort 5
- Parent communities: Peanut app, parenting Facebook groups: Cohort 3

## Gate 3 KPI Package
D30 retention ≥70% must be demonstrated across ≥3 cohorts before PROCEED decision.
Full package: retention curves + churn taxonomy + NPS + Affection Score + security audit + red team report.
